What is Computer Hardware?

Computer hardware refers to the physical components of a computer system. These are the tangible parts that you can touch and see. The major hardware components include:

  • Central Processing Unit (CPU)
  • Motherboard
  • Memory (RAM)
  • Storage Devices (HDD, SSD)
  • Power Supply Unit (PSU)
  • Input Devices (Keyboard, Mouse)
  • Output Devices (Monitor, Printer)
  • Peripheral Devices (External Storage, Networking Components)

Central Processing Unit (CPU)

The CPU is often referred to as the "brain" of the computer. It performs calculations and executes instructions to run programs.

Motherboard

The motherboard is the main circuit board that houses the CPU, memory, and other essential components. It provides the connections and sockets for other hardware components.

Memory (RAM)

Random Access Memory (RAM) is the temporary storage that the CPU uses for processing tasks. It is volatile memory, meaning it loses its data when the computer is turned off.

Storage Devices

Hard Disk Drive (HDD): Uses spinning disks to read/write data.
Solid State Drive (SSD): Uses flash memory for faster data access.

Power Supply Unit (PSU)

Converts electrical power from an outlet into a usable form for the computer. Powers all the internal components.

Input Devices

Devices used to input data into the computer (e.g., keyboard, mouse).

Output Devices

Devices used to output data from the computer (e.g., monitor, printer).

Peripheral Devices

Additional devices that provide extra functionality (e.g., external hard drives, network cards).

Motherboard

The central hub where all other components connect. Contains slots for the CPU, RAM, and expansion cards.

CPU

Installed on the motherboard. Covered with a heat sink and fan to prevent overheating.

RAM

Installed in the RAM slots on the motherboard. Determines the system's ability to handle multiple tasks simultaneously.

Storage

Hard Disk Drives (HDDs): Larger capacity, slower access speeds.
Solid State Drives (SSDs): Smaller capacity, faster access speeds.

Power Supply

Supplies power to all components. Typically located at the top or bottom of the case.

Expansion Cards

Installed in PCI slots on the motherboard. Include graphics cards, sound cards, network cards, etc.
