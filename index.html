Computer Hardware Components

Component Description
CPU (Central Processing Unit) The brain of the computer, responsible for processing instructions.
Motherboard The main circuit board that connects all components together.
RAM (Random Access Memory) Temporary memory that the CPU uses to store data for quick access.
GPU (Graphics Processing Unit) Handles rendering of images, video, and animations.
Storage (HDD/SSD) Stores all your data, files, and the operating system.
Power Supply Unit (PSU) Provides power to all components of the computer.
Cooling System Keeps the components cool to prevent overheating.
